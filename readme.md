# StockBroker Pro – Real-Time Stock Dashboard

StockBroker Pro is a frontend-only stock market dashboard built using React, Vite, and Tailwind CSS.  
It simulates live stock price updates and provides sorting and watchlist functionality without using any backend.
---
## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React
- Render (deployment)
